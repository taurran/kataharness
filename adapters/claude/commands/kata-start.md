---
description: Begin a new Kata run
argument-hint: "[goal or context for the run]"
---

Invoke the `kata-initiate` skill to start a new Kata run. Pass through: $ARGUMENTS. The goal text may name a **run-shape** — e.g. "debug my repo" (a `debug` run), a batch/best-of-N, or a version-up on an existing repo — which `kata-initiate` recognises and carries forward to `kata-bootstrap`. Do not improvise; if the skill is unavailable, say so.
